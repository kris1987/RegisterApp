RegisterApp
===========

 自我介绍系统

===============

如果只是想看效果，可以直接运行./RegisterApp.jar 
当然，需要你的电脑上面有JAVA环境

==============

现在程序里面的数据是从程序当前目录中的“members”文件中读取
数据格式如下 ：
> test1,id 1
> test2,id 2
> test3,id 3
> test4,id 4

程序会读取每一行，并且用逗号将字符串分隔，第一部分为用户的真实姓名，第二部分为用户的eoe id
